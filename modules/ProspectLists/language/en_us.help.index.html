Target Lists Home

Use this page to view existing lists of campaign targets.

The Target Lists Home page displays the following information:

  • A Search sub-panel where you can enter the target list name or target list type to search for an existing target list. To perform an advanced search using additional fields, click the Advanced Search tab. To customize save the search layout and results, click the Saved Search & Layout tab.
  • A list of existing target lists along with related information such as list type and assigned user.
  • To view target list details, click its name.
  • To update or delete multiple target records, select them from the list, and use the Mass Update sub-panel.
  • To export one or more target lists, select them from the list, click the Export link and choose Selected Records; to export all the lists on the page or the entire list, select Current Page and Entire List respectively.

A Shortcuts section that displays the following options:

  • Create Campaign. Click this option to create a new campaign.
  • Campaigns. Click this option to navigate back to the Campaigns Home page from a campaign's detail page.
  • Create Target List. Click this option to create a new list of targets.
  • Target Lists. Click this option to navigate back to the Target Lists page from the detail page of a target list .
  • Create Target. Click this option to create a new target.
  • Targets. Click this option to navigate back to the Targets list from a target's detail page.

A New Target List quick form that you can use to create a target list with only the required fields. You can enter additional details after you save the record.
