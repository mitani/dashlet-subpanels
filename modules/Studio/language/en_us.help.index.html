Studio

Use Studio to configure and edit drop-down lists, custom fields, layouts, and labels in Sugar modules.

You can select from the following options:

  • Edit a Module. Select this option to reconfigure a specific module.
  • Edit Drop Downs. Select this option to edit or create a drop-down list.
  • Configure Tabs. Select this option to hide a tab or to display a hidden tab.
  • Rename Tabs. Select this option to rename a tab.
  • Configure Group Tabs. Select this option to hide a tab or to display a hidden tab.
  • Edit Portal. Select this option to create and manage portal site.
  • Repair Custom Fields. Select this option to repair a custom field that you created in Sugar Suite.
  • Migrate Custom Fields. Select this option to import or export custom field definitions.
