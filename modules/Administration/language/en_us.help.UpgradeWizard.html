Module Loader

Use this page to upload custom modules, new language packs, and custom themes so that they are included in the current installation. When you create a module, you can save it as a zip file on your local machine. You can upload it into your Sugar application using the Module Loader.

To upload a module:

  1. In the Module field, enter the path to the module or click Browse to navigate to the location of the module zip file.
  2. Click Upload.
  3. Click Install to unzip the module file; to unload the module, click Delete Package.
  4. Click Commit to install the module.

After you install a module, you can disable it or uninstall it.

  • To disable the module, click Disable, accept the Sugar License Agreement, and then click Commit.
  • To uninstall the module, click Uninstall, accept the Sugar License Agreemen, and then click Commit.
