View Offline Clients

Use this page to view Sugar Offline Client users in your organization:

This page displays the following information:

  • A Search panel where you can enter the user name to search for a specific Offline Client user.
  • A list of existing Offline Client users in your organization.

To change the status of an offline client, use the Set Status column in the Offline Clients list.

To change the status of multiple Offline Client users, select them from the list and use the Mass Update panel. To enable them, select Active from the Offline Client Status drop-down list; to disable them, select Inactive from the drop-down list. Click Update to change the status.

To delete multiple Offline Client users, select them from the list and click Delete in the Mass Update panel.
