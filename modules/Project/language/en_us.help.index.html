Projects

Use the Projects module to create and manage projects for your organization. A project can be associated with multiple tasks and activities pertaining to one or more accounts and opportunities.


The Project Home page displays the following information:

  • A New Project Quick Form that you can use to create a project with only a few fields, including the required fields. You can enter additional information after you save the project.
  • A Search panel where you can enter the project name to search for it.
    • To find only projects assigned to you, select the Only my items box.
    • To perform an advanced search using additional fields, click the Advanced Search tab. To customize and save the search layout and results, click the Saved Search & Layout link.
  • A list of existing projects along with related information, such as estimated time and actual effort.
    • To view the details of a project, click the name in the Project list.
    • To update or delete multiple projects, select the projects from the list and use the Mass Update sub-panel.
  • A Shortcuts section that displays the following options:
    • Create Project. Click this option to create a new project. Alternatively, you can use the New Project quick form to create the project and add enter additional information after you save it.
    • Project List. Click this option to navigate back to the Projects Home page from a project's detail page.
    • Create Project Task. Click this option to create a new project task.
    • Project Tasks. Click this option to view a list of tasks for all your projects.
