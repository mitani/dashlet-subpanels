Calendar

Use the Calendar module to view and create calls, meetings, and tasks. The calendar displays your meetings, calls, and tasks scheduled for the day. The Day, Week, Month, Year, and Shared buttons display above the calender. To view your schedule for the week, month, or year, click the appropriate button.

Users can check the schedules of other users they plan to invite to a call or meeting. Click Shared and check the user's schedule in the shared calendar. Administrators can view as well as manage user schedules.

You can move backwards and forwards in the calendar- by one day, week, month or year. In the week, month or year format, you can click on any date to view that date in day format.

Other options that display on the Calendar Home page are:

  • A Search field to perform a global keyword search at the system level for text that matches the beginning of the keyword. In global search, the system searches the following modules: Accounts, Contacts, Leads, Opportunities, Quotes, Cases, Bug Tracker, and Projects. If you want to search only some of these modules, click the down arrow and select the modules from the list before you click GO.
  • A Shortcuts section that displays the following options:
    • Schedule Call. Click this option to schedule a call. Alternatively, click the appropriate hour in the left pane, select Schedule Call, enter the subject and click Save. You can then add additional information on the call's detail page.
    • Schedule Meeting. Click this option to schedule a meeting. Alternatively, click the appropriate hour in the left pane, select Schedule Meeting, enter the subject and click Save. You can then add additional information on the meeting's detail page.
    • Create Task. Click this option to create a task.
    • Calls. Click this option to view the status of calls that have been made.
    • Meetings. Click this option to view a list of past, present, and future meetings.
    • Tasks. Click this option to view a list of tasks assigned to you and to other users.
    • Import Calls. Click this option to import information on calls from an external application or file.
    • Import Meetings. Click this option to import information on meetings from an external application or file.
    • Import Tasks. Click this option to import information on tasks from an external application or file.
  • A Create Appointment quick form, located beneath Shortcuts, which you can use to schedule a new call or meeting. Enter the subject, select the date and start time, and click Save. You can add optional information after you create the appointment.
