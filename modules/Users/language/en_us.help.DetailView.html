Users Detail Page

View and manage user information such as home and email addresses, and the default user settings, locale settings, and PDF settings. Users can also view their access permissions for Sugar modules, their assigned roles, and assigned teams. If you log in as an administrator, you can duplicate a user record in order to create a similar new record.

  • Click Edit to change information such as name, password, email address, Sugar modules' layout, locale and PDF settings.
  • To duplicate the user record, click Duplicate. This button displays only when you log in as an administrator.
  • To change the user password, click Change Password, enter the new password, and click Save.
  • If you log in as a system administrator and want to send a system-generated password to a user, select the user from the list, and click Send Password.
  • To revert back to the default values of your preferences, HomePage, or Dashboard, click User Preferences, Homepage, or Dashboard respectively and click OK to confirm the action.
