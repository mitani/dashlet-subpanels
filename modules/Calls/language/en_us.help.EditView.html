Calls

Use the Calls page to create a call. Use the Search fields in the Add Invitees sub-panel to find and invite users to participate in the call. The schedules of all the participants display in the Scheduling sub-panel.

The Call List on the Calls Home page displays information such as the subject, contact, related item, due date, and user for each call. To view the details of a call, contact, or related item, click the appropriate Subject, Contact, and Related To link respectively. From the detail page, you can edit, duplicate, and delete information.

  • To create the call, enter required information such as the subject, start time and date, call type, and status.
  • To indicate the item record to which the call is related select the item, such as an Account or Opportunity, from the drop-down list and enter the record name in the adjoining field. You can also click Select to select the record from the Item list.
  • To set a default time for a reminder to alert users when the call is due, select the Reminder box. The system displays a pop-up reminder in the users' browser when the call is due.
  • To save the call information without sending out the invitation, click Save.
  • To send out invitations, click Send Invites.
  • To close the call and create a copy of it, click Close and Create New. The status of the closed call changes to Held; the status of the newly created call is, by default, set to Planned.
  • To exit the page without saving the information, click Cancel.
  • To view a list of calls or to search for a specific call, click Calls in the Shortcuts section.

Notes and Attachments

When you save the call, the Detail page displays the Notes sub-panel.

  • To create a note or attach a document to the call, click Create Note or Attachment. On the Notes page, enter the text in the Note field and specify the team that is allowed to view it.
  • To add an attachment, click Browse to locate the file. When you save the note, it is associated with the call and sent out along with the invitation.
  • To view a list of notes, click Notes in the Shortcuts panel of the Activities module.
