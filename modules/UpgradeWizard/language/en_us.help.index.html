Upgrade Wizard

Use this Wizard to upload new patches to upgrade your existing Sugar application. Before you begin this process, ensure that the config.php file for your installation is writable.
Previously installed upgrade packages are listed on the page.

  1. Click Next to perform System checks to ensure that you have the necessary file permissions and database privileges to upgrade the system. The status of System Checks changes to In Progress.
  2. Click Next to upload the upgrade files. When you obtain the Sugar upgrade .zip file from Sugar, you can save it to your local machine and upload it into Sugar using the Upgrade Wizard.

To upload from your local machine:
  1. Click Browse, navigate to the location of the Upgrade zip file, and select it.
  2. Click Upload Upgrade to upload the package to the Sugar application.
  3. Click Next to view the Preflight Check page.
    By default, the Upgrade Wizard Runs SQL option is selected as the database update method. If you ran the SQL queries manually, select Manual SQL Queries from the Database Update Method drop-down list and select the Check when SQL has been manually run box.
  4. Click Next to view the Commit Upgrade page.
  5. Click Next to view the Debrief screen and confirm that the upgrade has been installed.
  6. Click Done.
