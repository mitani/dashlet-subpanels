Campaign Wizard

The Campaign Wizard takes you through the process of creating a campaign and performing related tasks.

Enter the following information:

    Campaign Header

    • Name. Enter the campaign name.
    • Status. From the drop-down list, select the current state of the campaign.
    • Assigned to. Enter the name of the user who is assigned to the campaign. To select from the Users list, click Select. By default, it is assigned to you.
    • Type. If you are creating a non-email campaign, select the campaign type from the drop-down list.
    • Frequency. This field displays only for newsletter campaigns. From the drop-down list, select the intervals at which the newsletter will be emailed to targets.
    • Start Date. Enter the campaign start date; click the Calendar icon to select the date.
    • End Date. Enter the campaign end date; click the Calendar icon to select the date.
    • Description. Optionally, enter a brief description of the campaign.
    • Next. Click this button to enter the campaign budget information.

    Campaign Budget

    • Budget. Enter a numerical value for the campaign budget.
    • Actual Cost. Enter the actual cost of the campaign.
    • Expected Cost. Enter the expected cost of the campaign.
    • Expected Revenue. Enter the estimated revenue from the campaign.
    • Currency. From the drop-down list, select the currency used for the campaign.
    • Next. Click this button to enter the tracker URL information.

    Campaign Tracker URLs

    • Tracker Name. Enter a tracker name.
    • Opt-out link?. Select this box if you are creating an opt-out tracker that allows targets to unsubscribe from the newsletter mailing list.
    • Tracker URL. Enter the tracker URL. For example, www.companyName.com, and click Create Tracker. To create additional tracker URLs, click Create Tracker again.
    • Next. Click this button to view the next page, where you need to enter the target information for email and non-email campaigns. For newsletter campaigns, you need to enter the subscription list information.

    Target Lists

    Click Select to select an existing target list. To create a new target list, enter the name in the Target List field below, select the list type from the Target List Type drop-down list, and click Create.

    Subscription Information

    • Subscription List Name: Enter or select the subscription list name for targets who will receive the newsletter regularly.
    • Unsubscription List Name: Enter or select the unsubscription list name for targets who have chosen to opt-out of your mailing list.
    • Test list: Enter or select the test list name for targets who are the recipients of the test email that you send out to ensure it is successful.

    To create a custom list, click Allow Select and then click Select to search for targets and create the list.
    For a system-generated list, click Auto-Create and the system displays the name of the list in the field above the Select button.

    Save and Continue. Click this button to save the campaign information and begin the process of specifying other details such as the email settings.

    Finish. Click this button to save the campaign and specify the email settings at a later date.
