Cases

Use the Cases module to create and manage customer issues with your products.

The Cases Home page displays the following information:

  • A Search sub-panel where you can enter keywords such as the case number and subject to search for specific cases.
    • To restrain the search to cases related to items that are assigned to you, select the Only my items box.
    • To perform an advanced search using additional fields, click the Advanced Search tab.
    • To customize and save the search layout and results, click the Saved Search & Layout link on the Advanced Search tab.
    • A Search sub-panel where you can enter the subject or contact to search for related activities. To perform an advanced search using additional fields, click the Advanced Search tab. To view a column in the search result, move it to the Display Column list using the left arrow. You can also change the order in which the results display from Ascending to Descending. Enter a name for the search results in the Save this View as ? field and click Save.

  • A list of existing cases along with related information such as bugs, activities, and history.
    • To view the details of a case, click the name in the Case list.
    • To export information on one or more cases in .csv format to your local machine, select them in the list, click the Export link, choose Selected Records, and save the file. To export all the cases listed on the page or the entire list, select Current Page and Entire List respectively.
    • To merge duplicate cases into one record, select the records from the list and click Merge Duplicates, which is located above the list.
    • To update or delete multiple cases, select them from the list, and use the Mass Update sub-panel.

  • A Shortcuts section that displays the following options:
    • Create Case. Click this option to report a new case.
    • Cases. Click this option to navigate back to the Cases home page from a case's detail page.
    • Cases Reports. Select this option to view the pre-defined case reports that Sugar provides for your use.
    • Import. Select this option to import case data in a comma delimited, tab delimited, or custom delimited file.
    • A New Case quick form that you can use to create a case with only the required fields. You can enter additional details after you save the case.
