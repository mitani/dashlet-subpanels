Search and View Data

Use this page to search a data source by keywords.

  1. Click the tab of the data source connector that you want to use.
  2. Enter the keywords in the Search panel and click Search.
    The search results display within the tab for the data source connector.
  3. To view data in the data source record in a pop-up window, including additional data that might not be available in the List View, point the cursor on the down arrow adjacent to the Select button.
  4. From the search results that display below, select the data source record from which you want to merge data into the Sugar record.
  5. To merge the data, click Merge.
