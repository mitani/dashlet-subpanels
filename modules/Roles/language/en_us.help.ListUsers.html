Roles

Use this page to view and edit a role that is assigned to a user. You can assign multiple roles to a user.

  • To view the role assigned to a user, select the user from the Roles drop-down list. The assigned roles display in the Roles sub-panel below.
  • To view a list of existing roles, click Select in the Roles sub-panel and then click Search in the Search window.
  • To assign a role to the user, click the role from the Roles list in the Search window. The role is assigned to the user and is displayed in the Roles sub-panel.
  • To unassign a user from a Role, click the appropriate remRemove icon in the Roles sub-panel and click OK to confirm that you want to remove the relationship.
  • To edit a role, do the following:
    1. Click the Edit icon in the Roles sub-panel. The Create Role page displays on the screen.
    2. Click Save to display the Edit view where you can make the necessary changes to the role.
