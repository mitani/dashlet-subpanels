PDF Settings

Use this page to specify PDF settings when saving reports and quotes in PDF format.

In the Basic Settings section, specify the following information:

  • TCPDF and EZPDF. Select TCPDF to display international characters correctly in PDF files. If your organization does not require this capability, select EZPDF.
  • PDF Creator. The creator of the PDF files, which typically is an application such as Sugar.
  • PDF Author. The name of the user who created the PDF file.
  • Actual Header Logo. The image logo that displays in quotes and reports. The image that displays here is the logo that you specify in System Settings.
  • Upload New Logo. Click Browse to select a different Header Logo.
  • Header Logo Width (mm). The width of the logo in millimeters.
  • Header String. The text for the Header logo.
  • Header Title. The title associated with the logo.
  • Title. The title of the document.
  • Filename. The name of the PDF file.
  • Subject. The subject of the PDF file.
  • Keywords. The keywords associated with the document to help users find the document when they perform a keyword search.

In the Advanced Settings section, specify the following information:

  • Compression. Enable or disable page compression to reduce PDF file size. The default compression ratio is 2.
  • PDF Version. Enter the PDF version number. The default version is 1.7.
  • User Password. Optionally, specify a user password to ensure that only users who have the password can view PDF files.
  • Owner Password. Optionally, specify a master password to override the user password to view PDF files.
  • Cell Height Ratio. Specify the ratio of the cell height to the font height. The default height is 1.25.
  • Unit. Select a unit of measure, Millimeter, Point, Centimeter, or Inch, from the drop-down list. The default is Millimeter.
  • JPEG Quality. Enter the desired JPEG compression quality. The default value is 75.
  • Small Font Factor. Enter the scaling factor for small fonts.
  • Image Scale Ratio. Enter the ratio to scale images.

Click Save to save the settings.
