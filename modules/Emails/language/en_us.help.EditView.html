Create Archived Emails

Use the Create Archived Emails page to archive emails that you sent or received from your contacts and other users.
Copy the email into the Archived Email form. To associate it with a specific sugar record, select the related module from the drop-down list above and click Select to specify the record. The archived email is listed in the History panel of the related record.

To attach a file that is located on your machine, click Add File and navigate to the file location.

To attach a file that is located in the Sugar database, click Add Document and select the file from the Documents list.

To archive the email in plain text format, select the Edit Plain Text box located below Body and paste the text the text box that displays below.
